Запуск (из корня репозитория):

python3 webapp/app.py --index indexes/raw --host 0.0.0.0 --port 8080
```

В браузере http://127.0.0.1:8080/ — простая форма поиска и страница результатов (50 результатов на страницу).

Замечания:
- Приложение использует существующую логику загрузки словаря и постингов из `bin/search_cli.py`.
